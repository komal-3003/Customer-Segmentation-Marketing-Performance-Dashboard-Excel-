# Customer Segmentation Marketing Performance Dashboard Excel
## Project Objective
The objective of this project is to analyze customer behavior and marketing campaign performance by segmenting customers based on key attributes such as income, education, and purchasing patterns. The goal is to identify high-value customer segments and evaluate campaign effectiveness to support targeted marketing strategies.

## Project Description
- This project features an interactive Excel dashboard designed to track customer metrics and evaluate marketing performance.
- Using Pivot Tables, Pivot Charts, KPI cards, and slicers, the dashboard provides insights into customer segmentation, spending behavior, and campaign conversion rates. It enables users to dynamically explore how different customer groups contribute to revenue and respond to marketing campaigns.

## Key Performance Indicators (KPIs)
- Total Customers
- Total Revenue
- Average Customer Value (Revenue / Total Customers)
- Conversion Rate (Campaign Success Rate)

## Additional metrics include:
- Segment-wise revenue contribution
- Campaign acceptance rate
- Customer distribution by income and education
  
## Project Process
- 1.Data Collection
The dataset includes customer-level data such as:
  - Income
  - Education
  - Purchase behavior
  - Campaign responses
  - Spending patterns

- 2.Data Cleaning & Preparation
  - To ensure accurate analysis, the dataset was preprocessed:
  - Removed null and missing values
  - Checked and eliminated duplicate records
  - Standardized categorical fields (education, marital status, etc.)
  - Verified data types and corrected inconsistencies
  - The dataset required basic cleaning and validation before analysis.

- 3.Data Analysis
  - Segmented customers based on income, education, and spending behavior
  - Analyzed campaign performance across different segments
  - Calculated key KPIs such as average customer value and conversion rate

- 4.Dashboard Development
An interactive dashboard was built using Excel with:
  - KPI Cards for key metrics
  - Pivot Tables & Charts for segmentation analysis
  - Slicers for dynamic filtering
  - Visuals to track revenue contribution and campaign performance

## Project Insights
- High-income customers contribute a major share of total revenue

- Certain segments respond better to campaigns, highlighting opportunities for targeted marketing

## Conclusion
The dashboard provides a clear understanding of customer segmentation and marketing effectiveness. It helps businesses identify high-value customers, optimize campaign strategies, and improve overall marketing ROI through data-driven insights.
